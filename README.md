# cover-me

A Claude Code plugin marketplace with AI coding workflow tools.

## Install

```
/plugin marketplace add blawhi2435/cover-me
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `git-workflow` | Git commit and push workflow skills |
| `code-review` | Multi-angle code review skills |
| `standard-coding-style` | Coding standards and best practices |

## Usage

After adding the marketplace, install individual plugins:

```
/plugin install git-workflow@jerry-ai-tools
/plugin install code-review@jerry-ai-tools
/plugin install standard-coding-style@jerry-ai-tools
```

Update to latest versions:

```
/plugin marketplace update jerry-ai-tools
```
